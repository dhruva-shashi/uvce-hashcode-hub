Booksack

You were assigned the task of shifting \(N\) books from the entrance of UVCE to the library. You are enthusiastic in performing the task and decide to carry as many books as possible. But you are also aware of the fact that you cannot carry any weight greater \(W\) than in your sack. The goal is to find \(K\) books whose sum of weights does not exceed \(W\) and whose sum of weights is maximum.

Input Format
  • Each input data set is provided in a plain text file containing exclusively ASCII characters with lines terminated with a single '\n' character. When a single line contains multiple elements, they are separated by single spaces.
  • The first line of the data set contains two integers \(N\) and \(W\).
  • The second line consists of \(N\) integers \(a_1, a_2, a_3, ...., a_N\) in non-decreasing order.
Constraints
  • \(1 \le N \le 10^5\)
  • \(1 \le W \le 10^9\)
  • \(1 \le a_i \le W\)
Output Format
  • The output file should contain two lines.
  • The first line should contain an integer \(K, 0 \le K \le N,\) the number of books you decide to carry.
  • The second line should contain \(K\) integers, the indices of the books that you decide to carry.
